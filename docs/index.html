SecureWallet

Vitrine de preproduction du projet final DevSecOps : gouvernance Git, CI durcie, secrets chiffres, conteneurisation, SBOM et deploiement continu.

Architecture livree

Frontend statique frontend/ publie via GitHub Pages avec artefact ephemere.
Backend Node.js backend/ expose l'API Express et le healthcheck Vercel.
Image Docker Build multi-stage, utilisateur non-root, scan Trivy avant GHCR.
Secrets GitOps .github/secrets-prod.yaml chiffre uniquement les valeurs avec SOPS et age.

Chaine de validation

  • staging execute la CI complete a chaque push ou pull request.
  • main declenche la production seulement apres tests, Gitleaks, CodeQL, SBOM et scan image.
  • preprod sert de branche de previsualisation sans deploiement GitHub Pages.
  • Les jobs de deploiement utilisent des permissions minimales et un environnement production.

Controles securite

Shift-left local Hook pre-commit versionne dans scripts/pre-commit.sh et installable par scripts/install-hooks.sh.
Gitleaks Regle dediee aux jetons SECWALLET_ suivis de 24 caracteres majuscules ou chiffres.
CodeQL Echec du job si une alerte majeure ou de niveau error est detectee.
Interruption absolue Aucun continue-on-error sur les controles bloquants.
